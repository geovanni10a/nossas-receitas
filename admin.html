Sincronizacao

Wizard de configuracao do GitHub

Leitura funciona sem token. Para salvar no repositorio e sincronizar entre dispositivos, siga os 3 passos abaixo.

Criar token
  1. Crie um token pessoal para este repositorio e copie o valor gerado.
  2. Confira owner, repositorio e branch abaixo. Em forks, voce pode sobrescrever essa deteccao automatica.
  3. Cole o token, valide e confirme se o painel mostra o total de receitas e o destino correto.
